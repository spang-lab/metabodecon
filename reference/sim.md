# The Sim Dataset

A simulated dataset generated from the
[Blood](https://spang-lab.github.io/metabodecon/articles/Datasets.html#Blood)
dataset.

## Usage

``` r
sim
```

## Format

A `spectra` object consisting of 16 `spectrum` objects, where each
spectrum contains 2048 datapoints ranging from 3.60 to 3.29 ppm. For
details about `spectrum` and `spectra` objects see [Metabodecon
Classes](https://spang-lab.github.io/metabodecon/articles/Classes.html).
